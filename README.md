# Dineshkumar Anbalagan Portfolio

A clean personal portfolio website for a Java Backend Developer profile.

## Tech Used

- HTML
- CSS
- JavaScript

## Sections

- Home
- About
- Skills
- Projects
- Experience
- Contact

## How to Run Locally

Open `index.html` in your browser.

## How to Deploy on GitHub Pages

1. Create a GitHub repository named `dineshkumar-portfolio`
2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/`
3. Go to repository `Settings`
4. Click `Pages`
5. Select:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /root
6. Save

GitHub will give you a portfolio link.
